# Customer Purchase Behavior Analytics

> **An end-to-end data analytics project that transforms raw customer shopping data into actionable business insights using Python, SQL, and Power BI.**

This project explores customer purchasing behavior by performing data cleaning in Python, business analysis in SQL, and interactive dashboard development in Power BI. The analysis helps identify customer demographics, purchasing trends, product preferences, and sales patterns to support data-driven business decisions.

---

# Project Overview

Retail businesses generate thousands of customer transactions every day. Understanding who customers are, what they buy, how frequently they purchase, and which products perform best enables organizations to make informed marketing and sales decisions.

This project demonstrates a complete analytics workflow—from raw data to an interactive business dashboard.

---

# Business Objectives

* Analyze customer purchasing behavior
* Identify sales patterns across categories
* Study customer demographics
* Understand purchasing frequency
* Explore regional buying trends
* Build an interactive business dashboard
* Generate actionable business insights

# Data Preparation

The dataset was cleaned and prepared using Python.

### Data Cleaning

* Imported raw CSV dataset using Pandas
* Filled missing review ratings using the **median rating within each product category**
* Improved overall data quality before analysis

### Feature Engineering

* Removed unnecessary columns
* Prepared a cleaner dataset for SQL analysis and dashboard development
